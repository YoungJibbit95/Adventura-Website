Installation

Prototyp lokal starten

Adventura nutzt JDK 21 und den Gradle Wrapper des Game-Projekts. Die folgenden Befehle werden im `Game`-Verzeichnis ausgeführt.

Voraussetzungen

  • JDK 21
  • Gradle Wrapper aus dem Repository
  • Auf macOS eventuell `-XstartOnFirstThread` für LWJGL

Schnellstart

cd Game
./gradlew buildGame
./gradlew runLauncher
./gradlew runSingleplayer

Startbefehle

Launcher

./gradlew runLauncher

Öffnet die Swing-Oberfläche mit Seed, Render Distance, Preview Radius, Host, Port und Username.

Singleplayer

./gradlew runSingleplayer

Startet eine Offline-Welt mit lokaler Generation.

Server

./gradlew runServer

Startet den autoritativen Server.

Local Join

./gradlew joinLocal

Verbindet einen Client mit dem lokalen Server.

Direkte Modul-Commands

Server mit Seed

./gradlew :server:run --args="--port 25565 --seed 1337"

Nützlich für Multiplayer-Smoke-Tests und deterministische Weltstände.

Client Auto-Join

./gradlew :client:run --args="--auto-join --connect 127.0.0.1 --port 25565 --username Player --render-distance 8"

Verbindet ohne Main Menu direkt zum lokalen oder entfernten Server.

Client Argumente

--seed N --preview-radius N --render-distance N --connect HOST --port PORT --auto-singleplayer --auto-join

Performance

Bei Stottern während Chunk-Loading: Mesh Budget, Render Distance oder World Preview reduzieren. Optional Ambient AO, Shadows oder Water deaktivieren.
