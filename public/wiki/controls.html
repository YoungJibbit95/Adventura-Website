Steuerung & Commands

Aktuelle Tastenbelegung und Chat-Kommandos des Preview-Clients.

Bewegung & Gameplay

  • Mouse: look around
  • WASD: move
  • Space: jump / fly up
  • Left Ctrl: fly down
  • Left Shift: faster movement / sprint
  • Left mouse: break block
  • Right mouse: place block
  • Right mouse on harvestable blocks: gather berries or herbs
  • 1-9: select hotbar slot
  • Mouse wheel: switch hotbar slot

UI & Debug

  • E: crafting and inventory
  • O: settings
  • R: return to spawn
  • F1: toggle HUD
  • F3: toggle debug overlay
  • F4: cycle game mode
  • T: chat
  • /: chat with command prefix
  • Esc: pause/resume
